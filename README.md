## 📌 Overview  
A **Netflix UI Clone** built with pure **HTML & CSS**, replicating the look and feel of Netflix’s landing page. This project demonstrates responsive design, clean code practices, and attention to detail in front‑end development.

## 🚀 Features  
- 🎨 Pixel‑perfect Netflix‑style layout  
- 📱 Fully responsive design for mobile, tablet, and desktop  
- ⚡ Lightweight and fast loading (no frameworks)  
- 🧩 Modular CSS for easy updates  

## 🛠️ Tech Stack  
- **Frontend:** HTML5, CSS3  
- **Tools:** GitHub, VS Code
